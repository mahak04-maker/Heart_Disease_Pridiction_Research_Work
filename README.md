# ❤️ Heart Disease Prediction using Machine Learning

## 📌 Project Overview
This project predicts the likelihood of heart disease using machine learning algorithms. It is based on the UCI Heart Disease dataset and compares the performance of multiple classification models to identify the most accurate approach.

## 🎯 Objectives
- Predict the presence of heart disease using patient health data.
- Compare the performance of different machine learning algorithms.
- Improve prediction accuracy through data preprocessing and model evaluation.

## 📂 Dataset
- Dataset: UCI Heart Disease Dataset
- Features: Age, Sex, Chest Pain Type, Cholesterol, Blood Pressure, Maximum Heart Rate, Exercise Induced Angina, etc.
- Target:
  - 0 = No Heart Disease
  - 1 = Heart Disease
